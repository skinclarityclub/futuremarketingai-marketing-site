# LinkedIn Banner Image Creative Brief

## Specifications

- **Dimensions:** 1584 x 396px (LinkedIn recommended)
- **Format:** PNG or JPG, RGB color space
- **Safe zone:** Keep key text/elements within center 1200 x 300px (mobile crops edges)
- **File size:** Under 8MB

## Primary Text

**Main copy (Dutch):** AI Marketing Medewerker voor Bureaus

**Supporting copy:** 6 Skills | 1 Medewerker | 0 Personeel

## Visual Direction

### Color palette

| Element    | Color  | Hex     |
| ---------- | ------ | ------- |
| Background | Dark   | #050814 |
| Accent 1   | Cyan   | #00D4FF |
| Accent 2   | Purple | #A855F7 |
| Text       | White  | #FFFFFF |
| Subtle     | Muted  | #94A3B8 |

### Layout (left to right)

1. **Left third:** FutureMarketingAI wordmark/logo in white
2. **Center:** Main text "AI Marketing Medewerker voor Bureaus" in white, supporting text below in cyan
3. **Right third:** Product screenshot (dashboard or skill cards from website) or 6 skill icons arranged in a compact grid

### Typography

- **Primary font:** Inter (matches website)
- **Main text:** Bold, 36-42pt equivalent
- **Supporting text:** Regular, 18-24pt equivalent
- **Logo:** As per brand guidelines

### Style guidelines

- Professional, tech-forward aesthetic
- Dark background with subtle gradient or grid pattern (matches website dark theme)
- Cyan (#00D4FF) for accents, highlights, and supporting text
- Clean whitespace -- do not crowd elements
- Subtle glow or light effect behind main text for depth

### Do not include

- Stock photography of any kind
- Generic AI imagery (robots, brains, neural networks)
- Cluttered text or more than 2 lines of copy
- Bright/light backgrounds
- More than 3 colors beyond the defined palette
- QR codes or URLs (too small to read on banner)

## Production Notes

- **Tool:** Canva (Pro or free tier)
- **Template suggestion:** Start from blank custom dimensions (1584 x 396)
- **Export:** PNG for best quality, test on both desktop and mobile LinkedIn views
- **Mobile check:** The left and right ~190px may be cropped on mobile -- keep critical text centered

## Reference

- Website theme: futuremarketingai.nl (dark theme with cyan accents)
- Skill icons from website can be screenshot and placed as visual elements
- Dashboard screenshots from fma-app if available
