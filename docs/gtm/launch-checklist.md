# GTM Launch Checklist

Stap-voor-stap checklist voor de founding member launch. Werk van boven naar beneden -- elke sectie bouwt voort op de vorige.

## Referentie documenten

- [[linkedin-profile-copy]] -- Profiel teksten (headline, about, featured, experience)
- [[linkedin-banner-brief]] -- Banner ontwerp specificaties voor Canva
- [[demo-video-script]] -- Script voor demo video opname
- [[linkedin-content-calendar]] -- 2-weken content planning
- [[icp-checklist]] -- ICP kwalificatie criteria
- [[target-agencies]] -- Target agency tracking CSV
- [[outreach-templates]] -- 5-touch outreach sequentie
- [[call-script]] -- Demo call script
- [[founding-member-agreement]] -- Overeenkomst template

---

## Pre-Launch (eenmalig, doe dit eerst)

### LinkedIn Profiel

- [ ] Update LinkedIn headline met goedgekeurde tekst uit [[linkedin-profile-copy]]
  - NL: "Ik help marketingbureaus opschalen zonder personeel | AI Marketing Medewerker | Oprichter FutureMarketingAI"
- [ ] Update LinkedIn about-sectie (Nederlandse versie uit [[linkedin-profile-copy]])
- [ ] Update LinkedIn featured-sectie met founding member page link
- [ ] Update LinkedIn experience-sectie (FutureMarketingAI als huidige rol)
- [ ] Ontwerp LinkedIn banner in Canva volgens [[linkedin-banner-brief]]
  - Afmetingen: 1584x396px, donker thema (#050814), cyan accent (#00D4FF)
- [ ] Upload banner naar LinkedIn

### Demo Video

- [ ] Open Loom en bereid screen share voor
- [ ] Neem demo video op met [[demo-video-script]] (doel: 2:30-3:00)
- [ ] Bekijk opname en neem eventueel opnieuw op
- [ ] Upload demo video naar LinkedIn als native video
- [ ] Voeg demo video toe aan founding member pagina op de website

### Sales Navigator en Prospecting

- [ ] Abonneer op LinkedIn Sales Navigator Core (~EUR 80/mo)
- [ ] Bouw NL target lijst (30 bureaus):
  - Zoek: Industry = Marketing Services, Geography = Netherlands, Size = 2-50
  - Evalueer elk bureau tegen [[icp-checklist]] (score 8+ = toevoegen)
- [ ] Bouw UK target lijst (20 bureaus):
  - Zoek: Industry = Marketing Services, Geography = United Kingdom, Size = 2-50
  - Evalueer elk bureau tegen [[icp-checklist]] (score 8+ = toevoegen)
- [ ] Vul [[target-agencies]] CSV met alle 50 bureaus
  - Alle kolommen invullen: naam, contact, LinkedIn URL, website, grootte, type, ICP score

### Booking Setup

- [ ] Stel Cal.com of Calendly in met 15-minuten demo slots
- [ ] Voeg booking link toe aan LinkedIn featured-sectie
- [ ] Test booking link (maak een test-afspraak)

---

## Launch Week (week 1 van outreach)

### Dag 1-2: Content en eerste batch

- [ ] Publiceer eerste LinkedIn post uit [[linkedin-content-calendar]]
- [ ] Start Touch 1 op eerste batch (10 bureaus):
  - Comment op hun recente LinkedIn post (zie [[outreach-templates]])
  - Update stage naar "warming" in [[target-agencies]]

### Dag 3-4: Tweede batch en connecties

- [ ] Publiceer volgende LinkedIn post uit content calendar
- [ ] Start Touch 1 op tweede batch (10 bureaus)
- [ ] Stuur Touch 2 (connection request) naar eerste batch
- [ ] Update stages in [[target-agencies]]

### Dag 5: Voortgang en waarde

- [ ] Publiceer LinkedIn post
- [ ] Start Touch 1 op derde batch (10 bureaus)
- [ ] Stuur Touch 2 naar tweede batch
- [ ] Stuur Touch 3 (value message) naar geaccepteerde connecties uit batch 1
- [ ] Update alle stages in [[target-agencies]]

---

## Ongoing (week 2-4)

### Dagelijkse taken

- [ ] Post 3-5x per week op LinkedIn per [[linkedin-content-calendar]]
- [ ] Werk alle batches door de 5-touch sequence (zie [[outreach-templates]])
- [ ] Reageer op alle inkomende berichten binnen 4 uur
- [ ] Update [[target-agencies]] na elke touch

### Wekelijkse taken

- [ ] Review voortgang in [[target-agencies]]:
  - Hoeveel in warming/connected/conversation/call-booked?
  - Hoeveel lost? Waarom?
- [ ] Start Touch 1 op volgende batches (10 bureaus per batch) tot alle 50 zijn bereikt
- [ ] Boek en voer demo calls uit met [[call-script]]
- [ ] Stuur founding member agreements naar geinteresseerde bureaus ([[founding-member-agreement]])
- [ ] Follow up op verstuurde agreements binnen 48 uur

### Progress tracking

Houd bij in [[target-agencies]]:

| Metric                        | Doel     | Huidige stand |
| ----------------------------- | -------- | ------------- |
| Totaal targets                | 50       |               |
| Connection requests verstuurd | 50       |               |
| Connecties geaccepteerd       | 35 (70%) |               |
| DM conversations gestart      | 15 (43%) |               |
| Demo calls geboekt            | 8 (53%)  |               |
| Pitched                       | 5+       |               |
| Signed                        | 3-5      |               |

## Milestone: Founding Members Signed

**Doel: 3-5 founding members ondertekend**

- [ ] Minimaal 3 bureaus in stage "signed" in [[target-agencies]]
- [ ] Alle ondertekende agreements opgeslagen
- [ ] Onboarding gepland voor elke founding member
- [ ] Founding member Slack/WhatsApp kanaal aangemaakt
- [ ] Eerste maandelijkse strategie-call ingepland

### Als je onder 3 signed zit na 50 targets:

1. Analyseer conversion funnel: waar vallen prospects af?
2. Update call script en outreach templates op basis van feedback
3. Bouw tweede lijst van 25-50 extra targets
4. Overweeg alternatieve prospecting bronnen:
   - DDMA ledenlijst (360+ organisaties)
   - Emerce 100 lijst
   - Clutch.co / Sortlist directories
   - Google "marketing bureau [stad]"
