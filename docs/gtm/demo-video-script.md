# Demo Video Script

**Target duration:** 2:30 - 3:00
**Recording tool:** Loom (screen + voiceover, no face cam needed)
**Tone:** Direct, practical, Dutch cultural directness -- no hype, no superlatives

---

## Section 1: Hook (0:00 - 0:15)

**Voiceover:**

> "Je bureau gebruikt 15 tools maar je kunt alsnog niet opschalen. Wat als je een AI medewerker had die ze allemaal runt?"

**On screen:** Quick montage of tool logos (HubSpot, Hootsuite, Canva, Google Analytics, Mailchimp, etc.) scattered across a cluttered desktop. Then: a clean transition to the FutureMarketingAI logo on dark background.

---

## Section 2: Problem (0:15 - 0:45)

**Voiceover:**

> "Als bureau-eigenaar herken je dit. Je team jongleert met 15 verschillende tools. Content planning in het ene systeem, social media scheduling in het andere, rapportages handmatig in Excel."

**On screen:** Split screen showing multiple tool windows open simultaneously, cursor jumping between tabs.

> "Je wilt groeien, maar je kunt niet genoeg mensen vinden. Of betalen. Een junior marketeer kost je EUR 3.000 tot 5.000 per maand -- als je er een kunt vinden."

**On screen:** Job posting for "Junior Online Marketeer" with salary range highlighted.

> "En ondertussen besteed je uren aan handmatige klantrapportages die niemand leest."

**On screen:** Spreadsheet with manual data entry, clock showing hours ticking by.

---

## Section 3: Solution Walkthrough (0:45 - 2:15)

**Voiceover:**

> "Dit is de AI Marketing Medewerker. Zes inplugbare skills. Een digitale collega die 24/7 werkt."

**On screen:** FutureMarketingAI dashboard showing the 6 skill cards. Brief pause to let viewer absorb.

### Skill Demo 1: Content Creator (0:55 - 1:25)

**Voiceover:**

> "De Content Creator skill. Je geeft een klant-briefing en de AI doet het onderzoek, schrijft merkgebonden content, en levert publicatie-klare stukken. In minuten, niet dagen."

**On screen:** Screen capture direction:

1. Show a client brief being entered (topic, tone, audience)
2. Show the AI processing (research phase, writing phase)
3. Show the finished content output with formatting
4. Highlight: "Van briefing tot publicatie in 4 minuten"

### Skill Demo 2: Social Media Manager (1:25 - 1:50)

**Voiceover:**

> "De Social Media Manager. Posts worden automatisch gepland en gepubliceerd op alle platformen van je klant. Geen handmatig inplannen meer."

**On screen:** Screen capture direction:

1. Show content calendar view with scheduled posts
2. Show multi-platform publishing (LinkedIn, Instagram, Facebook)
3. Show analytics dashboard with engagement metrics
4. Highlight: "3 platformen, 1 klik"

### Skill Demo 3: Reporting (1:50 - 2:15)

**Voiceover:**

> "En rapportage. Elke maand automatisch een klantrapport met de belangrijkste metrics, trends, en aanbevelingen. Geen Excel meer. Geen copy-paste."

**On screen:** Screen capture direction:

1. Show automated report generation triggered
2. Show the generated report: charts, metrics, insights
3. Show export options (PDF, email to client)
4. Highlight: "Van 4 uur handwerk naar 2 minuten"

## Production Notes

### Recording

- **Tool:** Loom -- screen recording with voiceover
- **No face cam:** Screen content is the focus
- **Resolution:** 1920x1080 minimum
- **Audio:** Use a decent microphone. Quiet room. No background music during voiceover.
- **Voiceover language:** Dutch (primary version). English version can be recorded separately if needed for UK outreach.

### Editing

- Keep cuts tight -- no dead air, no "uhh" moments
- Use Loom's built-in trim feature for basic edits
- Add text overlays for key metrics in Section 4 (Loom supports this)
- No fancy transitions -- clean cuts only

### Screen content preparation

- Before recording, prepare the dashboard/n8n with realistic data
- Use SKC pilot data where possible (real results are more convincing)
- If actual dashboard is not ready, use the website demo pages as fallback
- Clear all browser bookmarks and notifications before recording

### Distribution plan

1. **LinkedIn:** Native upload (not a YouTube link -- native gets 5x more reach)
2. **Website:** Embed on futuremarketingai.nl/founding-member page
3. **DM outreach:** Attach in Touch 4 of the outreach sequence
4. **Email:** Include link in follow-up emails after demo calls

### Authenticity over polish

- This is a founder demo, not a commercial
- Imperfect is fine -- authenticity builds trust with Dutch buyers
- Do not add stock music, fancy intros, or logo animations
- The content (working product, real results) does the selling
