2. Create a temporary Node.js script (e.g., `scripts/generate-placeholders.js`) that:

**For .webp poster images:**

- Use the `sharp` package (`npx` or install temporarily) to generate a 640x360 dark solid-color image (#050814) saved as .webp
- If sharp is not available, use the `canvas` package
- If neither is available, write a minimal valid WebP file using raw bytes. A minimal valid WebP file needs: RIFF header (4 bytes) + file size (4 bytes) + WEBP marker (4 bytes) + VP8 chunk. Use a pre-encoded base64 string of a tiny valid dark WebP.
- The posters should be plain dark rectangles matching #050814 — no text needed. They blend with the site background naturally.

**For .mp4 video files:**

- Write a minimal valid MP4 container using hardcoded bytes. The smallest valid MP4 needs: ftyp box + moov box (with minimal track/media atoms).
- Use a pre-encoded base64 string of a minimal valid 1x1 or small MP4 (~600 bytes).
- These files are never auto-downloaded (`preload="none"`) — they only need to be valid if explicitly played.

3. Generate all 8 files (4 services x 2 file types):
