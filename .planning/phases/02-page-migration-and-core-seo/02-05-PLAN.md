  3. AI training bots (block): `GPTBot`, `ClaudeBot`, `CCBot`, `Google-Extended`, `anthropic-ai` -- disallow `/`
- Include `sitemap: '${SITE_URL}/sitemap.xml'`
- Export default function returning `MetadataRoute.Robots`
