- `bg-gradient-to-br from-gray-900 via-blue-900 to-purple-900` → `bg-bg-deep`

**FloatingNav.tsx (~382 lines):**

This is a floating overlay navigation, likely with glassmorphism:

- `bg-slate-950/90 backdrop-blur-xl` or similar → `bg-bg-surface/95 border border-border-primary` (remove backdrop-blur)
- `bg-gradient-to-r from-blue-*` buttons → `bg-accent-system text-bg-deep`
- `text-blue-400` active → `text-accent-system`
- `text-slate-400` inactive → `text-text-secondary`
- `hover:text-white` → `hover:text-text-primary`
- `bg-blue-500/10` active background → `bg-accent-system/10`
- `border-white/10` → `border-border-primary`
- `rounded-xl/2xl` → `rounded-sm`
- Remove all `backdrop-blur-*`
- Any dropdown/popover bg: `bg-slate-900/95` → `bg-bg-elevated border border-border-primary`

**CookieConsent.tsx (~153 lines):**

Per 09-RESEARCH.md Pitfall 5: react-cookie-consent uses INLINE STYLES that override Tailwind.

- Use the library's `style` / `buttonStyle` / `declineButtonStyle` props with JS style objects:
  - Container: `style={{ background: '#111520', borderTop: '1px solid rgba(0,212,170,0.2)', color: '#E8ECF4' }}`
  - Accept button: `buttonStyle={{ background: '#00D4AA', color: '#0A0D14', borderRadius: '0.125rem', fontWeight: 600 }}`
  - Decline button: `declineButtonStyle={{ background: 'transparent', border: '1px solid rgba(90,99,120,0.3)', color: '#9BA3B5', borderRadius: '0.125rem' }}`
- Also update any Tailwind classes on the wrapper: old colors → Living System tokens
- `text-blue-*` → `text-accent-system`
- Link colors inside banner → `text-accent-system`
