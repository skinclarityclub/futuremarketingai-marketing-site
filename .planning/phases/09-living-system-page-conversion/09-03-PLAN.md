- `text-blue-100/80` → `text-text-muted`
- `text-blue-100` → `text-text-secondary`

- Card body: `text-blue-100/80` → `text-text-muted`

- **Remove** CalendlyModal import if no longer used
- **Remove** useState import if no longer used (check if other state remains)

- `text-blue-400` → `text-accent-system`
- `text-purple-400` / `text-violet-400` → `text-accent-human`

**8. Borders and shadows:**

- `border-white/10` → `border-border-primary`
