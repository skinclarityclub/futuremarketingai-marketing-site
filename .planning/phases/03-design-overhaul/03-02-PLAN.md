## Objective

Create the 5 shared Living System components that will be used across all pages: SystemPanel (replaces GlassCard), StatusIndicator (pulsing dots), MetricDisplay (large monospace numbers), CTAButton (consistent CTA with Calendly), SectionContainer (layout wrapper). All components use the new design tokens from Plan 03-01.

## Tasks

### Task 1: Create SystemPanel Component

**Files:** Create `src/components/common/SystemPanel.tsx`, Modify `src/components/common/index.ts`

Create SystemPanel with:

- Props: children, className, scanLines (boolean), hoverAccent (boolean), onClick, id
- Uses forwardRef
- Base: `bg-bg-surface border border-border-primary rounded-sm`
- Optional scan-lines CSS class
- Optional hover accent: `hover:border-l-2 hover:border-l-accent-system hover:bg-bg-elevated`
- Accessible: role="button" + tabIndex + onKeyDown when onClick provided
- Export from index.ts

**Commit:** `feat: add SystemPanel component — Living System primary container`

### Task 2: Create StatusIndicator Component

**Files:** Create `src/components/common/StatusIndicator.tsx`, Modify `src/components/common/index.ts`

Create StatusIndicator with:

- Props: variant (active/attention/inactive/system), label (string), pulse (boolean), size (sm/md/lg)
- Colored dot with optional pulse animation
- Optional monospace label (font-mono text-xs uppercase)
- Export from index.ts

**Commit:** `feat: add StatusIndicator component — pulsing status dots with monospace labels`

### Task 3: Create MetricDisplay Component

**Files:** Create `src/components/common/MetricDisplay.tsx`, Modify `src/components/common/index.ts`

Create MetricDisplay with:

- Props: value (string), label (string), accent (system/human/active), prefix, suffix
- Large monospace number: `font-mono text-3xl md:text-4xl font-bold`
- Small monospace label: `font-mono text-xs uppercase tracking-wider text-text-muted`
- Export from index.ts

**Commit:** `feat: add MetricDisplay component — large monospace numbers with labels`

### Task 4: Create CTAButton Component

**Files:** Create `src/components/common/CTAButton.tsx`, Modify `src/components/common/index.ts`

Create CTAButton with:

- Props: variant (primary/secondary), size (sm/md/lg), arrow (boolean), calendly (boolean), calendlyUrl, href, onClick, ariaLabel
- Primary: `bg-accent-system text-bg-deep`, sharp corners (rounded-sm)
- Secondary: transparent + border, hover fills
- Calendly integration: opens CalendlyModal with dark theme params
- Default Calendly URL: `https://calendly.com/futureai/strategy-call?background_color=111520&text_color=e8ecf4&primary_color=00D4AA`
- Renders as `<a>` for href, `<button>` otherwise
- Import ArrowRight from lucide-react
- Export from index.ts

**Commit:** `feat: add CTAButton component — consistent CTA with Calendly modal integration`
