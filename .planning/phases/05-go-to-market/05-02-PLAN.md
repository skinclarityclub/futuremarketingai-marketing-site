Create three files in docs/gtm/:

**icp-checklist.md** -- ICP qualification criteria with scoring:

Scoring criteria (each 0-2 points, total 0-12):

1. **Team size** (3-12 FTE = 2, 13-25 = 1, else 0)
2. **Focus** (performance/growth/content = 2, mixed = 1, branding-only = 0)
3. **LinkedIn activity** (founder posts weekly = 2, monthly = 1, inactive = 0)
4. **Multi-client** (serves 5+ clients = 2, 2-4 = 1, single-client = 0)
5. **Digital maturity** (uses 3+ marketing tools = 2, 1-2 = 1, none = 0)
6. **Geography** (NL = 2, UK = 1, other = 0)

Qualification threshold: 8+ = hot prospect, 5-7 = warm, below 5 = skip.

Anti-ICP signals (auto-disqualify):

- Creative/branding-only agency (no performance focus)
- One-person freelancer (no scaling need)
- Already has in-house AI team
- Enterprise (50+ employees, different buying process)

Include Obsidian frontmatter.

**target-agencies.csv** -- Tracking spreadsheet template:

Headers (in this order):
Agency Name, Contact Name, Contact Title, LinkedIn URL (person), LinkedIn URL (company), Website, City, Country, Team Size, Focus Type, ICP Score, Stage, Last Touch Date, Next Action, Next Action Date, Demo Call Date, Objections, Notes

Stage values: prospect | warming | connected | dm-sent | conversation | call-booked | call-done | pitched | negotiating | signed | lost

Pre-populate with 3 example rows (fictional Dutch agencies) to demonstrate usage. Leave remaining rows empty for manual population.

**outreach-templates.md** -- Complete 5-touch LinkedIn outreach sequence:

For each touch, provide Dutch AND English templates with [placeholder] tags:

**Touch 1 (Day 0): Warm engagement**

- Action: Comment on prospect's recent LinkedIn post
- Template comment (NL): Substantive 2-3 sentence comment adding insight, NOT "Goed punt!"
- Template comment (EN): Same approach
- Rule: Must reference specific content from their post

**Touch 2 (Day 2-3): Connection request**

- Template (NL): "Hoi [naam], ik zag je post over [onderwerp] -- herkenbaar. Ik bouw een AI marketing medewerker specifiek voor bureaus zoals [bureau naam]. Leuk om te connecten!"
- Template (EN): "Hi [name], saw your post about [topic] -- resonated with me. Building an AI marketing employee specifically for agencies like [agency]. Would be great to connect."
- Rule: Under 300 characters, no pitch

**Touch 3 (Day 5-7): Value message**

- Template (NL): Share insight + ask about their AI usage
- Template (EN): Same approach
- Rule: No pitch yet, start a conversation

**Touch 4 (Day 10-14): Soft pitch**

- Template (NL): Reference conversation, mention 10 founding member spots, EUR 997/mo, demo offer
- Template (EN): Same approach
- Rule: Attach demo video, 15 min no-obligation

**Touch 5 (Day 18-21): Follow-up / close the loop**

- Template (NL): If no response -- urgency with remaining spots. If responded but not committed -- address objection
- Template (EN): Same approach
- Rule: Final touch, respect if no interest

Include timing guidance and the 3-Touch Rule (comment, reply, connect) before pitching.
Include objection handling section with Dutch+English responses for:

1. "EUR 997/mo is veel" (price)
2. "AI kwaliteit is niet goed genoeg" (quality)
3. "Ik heb geen tijd" (time)
4. "Wat als ik stop?" (commitment)
5. "Ik wil eerst resultaten zien" (proof)

Use Obsidian frontmatter.
