**Create `src/components/chatbot/ChatWidget.tsx`:**

Props interface:

```ts
interface ChatWidgetProps {
  mode: 'floating' | 'embedded'
  personaId: string
  personaName?: string
  personaAvatar?: string
  suggestedPrompts?: string[]
  height?: string // embedded mode only, e.g., '500px'
  messageLimit?: number // demo limit, default 15
}
```

Implementation:

1. **Hook integration:**
   - Call `usePersonaChat(personaId)` to get `messages`, `sendMessage`, `status`, `messageCount`, `isAtLimit`
   - Call `useChatbotStore()` to get UI state: `isOpen`, `isMinimized`, `hasUnread`, `open`, `close`, `minimize`, `toggle`, `markRead`

2. **Send handler:**

   ```ts
   const handleSend = (text: string) => {
     if (isAtLimit) return
     sendMessage({ text })
   }
   ```

3. **Floating mode layout:**

   ```tsx
   if (mode === 'floating') {
     return (
       <>
         <FloatingButton
           onClick={toggle}
           hasUnread={hasUnread}
           isOpen={isOpen}
         />
         <AnimatePresence>
           {isOpen && !isMinimized && (
             <motion.div
               initial={{ opacity: 0, y: 20, scale: 0.95 }}
               animate={{ opacity: 1, y: 0, scale: 1 }}
               exit={{ opacity: 0, y: 20, scale: 0.95 }}
               transition={{ duration: 0.25, ease: 'easeOut' }}
               className="fixed z-50 right-6 bottom-24 lg:right-20 lg:top-[10vh] lg:bottom-auto
                          w-[calc(100vw-3rem)] max-w-[420px] h-[70vh] max-h-[600px]
                          bg-bg-surface/95 backdrop-blur-xl
                          border border-border-primary rounded-2xl
                          shadow-2xl shadow-black/40
                          flex flex-col overflow-hidden"
               role="dialog"
               aria-label={`Chat with ${personaName || 'assistant'}`}
               aria-modal="true"
             >
               <ChatHeader ... />
               <ChatMessages ... />
               {showPrompts && <SuggestedPrompts ... />}
               <ChatInput ... />
             </motion.div>
           )}
         </AnimatePresence>
       </>
     )
   }
   ```

4. **Embedded mode layout:**

   ```tsx
   return (
     <div
       style={{ height: height || '500px' }}
       className="flex flex-col bg-bg-surface/95 border border-border-primary rounded-2xl overflow-hidden"
     >
       <ChatHeader mode="embedded" ... />
       <ChatMessages ... />
       {showPrompts && <SuggestedPrompts ... />}
       <ChatInput ... />
     </div>
   )
   ```

5. **Suggested prompts visibility:**
   - Show `SuggestedPrompts` only when `messages.length === 0` and `suggestedPrompts` prop is provided
   - Pass `onSelect={handleSend}` so clicking a prompt sends it immediately
   - Hide when `isAtLimit`

6. **Focus management (floating mode):**
   - When panel opens, focus the textarea in ChatInput (pass ref down or use `useEffect` with querySelector)
   - When Escape is pressed inside the panel, close it

7. **Demo limit handling:**
   - When `isAtLimit`, show a banner above ChatInput: "Demo limit reached. Book a call to see the full experience." with a CTA link styled as `text-accent-system`
   - Disable ChatInput when at limit

8. **Imports:**
   - `AnimatePresence, motion` from `framer-motion` — ONLY for panel enter/exit animation, nothing else
   - All child components from this directory
   - `usePersonaChat` from hooks
   - `useChatbotStore` from stores
