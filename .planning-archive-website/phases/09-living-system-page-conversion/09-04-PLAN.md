**AboutPage.tsx (~199 lines):**

Known old patterns (from 09-RESEARCH.md): `bg-purple-500/10 border-purple-500/20`, `bg-white/5 backdrop-blur`, `text-blue-100`, page wrapper gradient.

Convert:

- Page wrapper gradient → `bg-bg-deep`

- Heading gradients → `bg-gradient-flow bg-clip-text text-transparent`
- `rounded-xl/2xl` → `rounded-sm`
- Remove `backdrop-blur`
- Any CTA buttons → CTAButton if linking to Calendly, otherwise update colors inline

**ContactPage.tsx (~333 lines):**

Contact page likely has a form, contact details cards, and possibly a map or social links.

Convert:

- Page wrapper → `bg-bg-deep`

- Form inputs: likely `bg-white/5 border-white/10` → `bg-bg-elevated border border-border-primary rounded-sm`
- Input focus states: `focus:border-blue-500` → `focus:border-accent-system`
- `text-blue-400` → `text-accent-system`
- Body text → `text-text-secondary` / `text-text-muted`
- Button/submit: gradient → `bg-accent-system text-bg-deep` or use CTAButton
- Social link icons: old blue → `text-accent-system`
- `rounded-xl/2xl` → `rounded-sm`
- Remove `backdrop-blur`

Preserve all Framer Motion animations.
