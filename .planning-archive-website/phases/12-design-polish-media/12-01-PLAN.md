   - Import OrbitVisual from './OrbitVisual'
   - Lazy-load Spline: `const Spline = lazy(() => import('@splinetool/react-spline'))`
   - Props: `{ sceneUrl: string; className?: string }`

3. Create `src/components/common/ScrollReveal.tsx`:
   - Import motion from 'framer-motion'
   - Import useMotionSafe from '../../hooks/useReducedMotion'
   - Props interface: `{ children: React.ReactNode; delay?: number; direction?: 'up' | 'left' | 'right' | 'none'; className?: string; as?: keyof JSX.IntrinsicElements }`
   - Default: delay=0, direction='up'
   - Compute initial based on direction:
     - 'up': { opacity: 0, y: 30 }
     - 'left': { opacity: 0, x: -30 }
     - 'right': { opacity: 0, x: 30 }
     - 'none': { opacity: 0 }
   - Animate target: { opacity: 1, y: 0, x: 0 }
   - Wrap in useMotionSafe: `const props = useMotionSafe({ initial, whileInView: animate, viewport: { once: true, amount: 0.2 }, transition: { duration: 0.6, delay, ease: 'easeOut' } })`
   - Return `<motion.div {...props} className={className}>{children}</motion.div>`

4. Create `src/components/common/ProductMedia.tsx`:
   - Import React and useReducedMotion from 'framer-motion'
   - Props: `{ videoSrc: string; posterSrc: string; alt: string; className?: string }`
   - If useReducedMotion() returns true, render `<img src={posterSrc} alt={alt} className="rounded-card w-full" loading="lazy" />`
   - Otherwise render `<video autoPlay muted loop playsInline poster={posterSrc} className="rounded-card w-full" preload="none"><source src={videoSrc} type="video/mp4" /></video>`
   - Pass className through to the wrapping element

Do NOT export these from src/components/common/index.ts yet — Plan 02/03 will import directly since index.ts is a shared file.

<task type="auto">
  <name>Task 2: Create useTilt hook with desktop-only guard</name>
  <files>src/hooks/useTilt.ts</files>
  <action>
Create `src/hooks/useTilt.ts`:

- Import useRef, useCallback, useState from 'react'
- Export `function useTilt(maxTilt = 8)`:
  - `const ref = useRef<HTMLDivElement>(null)`
  - `const [style, setStyle] = useState<React.CSSProperties>({ transform: 'perspective(800px) rotateX(0deg) rotateY(0deg)', transition: 'transform 0.15s ease-out' })`
  - `handleMouseMove` callback:
    - Early return if `!ref.current`
    - Get bounding rect
    - Calculate x = (clientX - rect.left) / rect.width - 0.5
    - Calculate y = (clientY - rect.top) / rect.height - 0.5
    - setStyle with rotateX(-y _ maxTilt) and rotateY(x _ maxTilt), transition 0.15s
  - `handleMouseLeave` callback:
    - Reset to rotateX(0) rotateY(0) with transition 0.4s (slower snap-back)
  - Return `{ ref, style, handleMouseMove, handleMouseLeave }`

- Add a `@media (hover: hover) and (pointer: fine)` guard comment explaining that callers should wrap usage in a desktop-only check (e.g., using matchMedia or the existing useIsMobile hook). The hook itself returns values unconditionally — the caller decides whether to apply them.

This keeps the hook pure and testable. Plan 02/03 will show the desktop-only guard pattern when applying.
