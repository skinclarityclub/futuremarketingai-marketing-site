## Objective

Replace the indigo/violet/pink glassmorphism design system with the "Living System" teal/amber palette. Update Tailwind config (colors, fonts, shadows, gradients), CSS custom properties, and add Space Grotesk font. This is the foundation all other plans depend on.

## Tasks

### Task 1: Update Tailwind Config — New Color Palette

**Files:** Modify `tailwind.config.js`

1. Replace the entire `colors` block with Living System palette:
   - Background: `bg-deep` (#0A0D14), `bg-surface` (#111520), `bg-elevated` (#1A1F2E)
   - Accents: `accent-system` (#00D4AA teal), `accent-human` (#F5A623 amber)
   - Status: `status-active` (#22C55E), `success`, `warning`, `error`, `info`
   - Text: `text-primary` (#E8ECF4), `text-secondary` (#9BA3B5), `text-muted` (#5A6378)
   - Border: `border-primary` (rgba 255,255,255,0.06), `border-accent` (rgba 0,212,170,0.3), `divider`

2. Replace `fontFamily` block:
   - sans: Inter stack
   - display: Space Grotesk, Inter
   - mono: JetBrains Mono, Consolas, Monaco

3. Replace `boxShadow` block with teal/amber glow effects

4. Replace `backgroundImage` block with system/human/flow gradients

5. Add new animations: `glow-pulse`, `float`, `slide-up`, `slide-down`, `fade-in`, `status-pulse`, `data-flow`

6. Run `npm run dev` to verify no build errors

**Commit:** `refactor: replace indigo/violet/pink palette with Living System teal/amber design tokens`
