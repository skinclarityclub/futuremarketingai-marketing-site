**2. `src/lib/chatbot/tools/ecommerce-tools.ts`** — Export `ecommerceTools` record

Import `PRODUCT_CATALOG` from `../knowledge/ecommerce-kb`.

Create 4 tools:

- `search_products`: inputSchema `{ query: z.string().optional(), skinType: z.enum(['dry', 'oily', 'combination', 'normal', 'sensitive']).optional(), concern: z.string().optional(), limit: z.number().min(1).max(5).default(3) }`. Execute filters PRODUCT_CATALOG by matching skinType, concern (substring match in concerns array), or query (substring match in name/description). Return `{ products: filtered.slice(0, limit) }`.
- `get_product_details`: inputSchema `{ productId: z.string() }`. Execute finds product by id in PRODUCT_CATALOG, returns full product object or `{ error: 'Product not found' }`.

Execute builds routine by selecting appropriate products from catalog for the skin type. Return `{ morning?: ProductStep[], evening?: ProductStep[] }` where ProductStep has `{ step: string, product: { id, name, price } }`.
- `add_to_cart_suggestion`: inputSchema `{ productId: z.string(), quantity: z.number().default(1) }`. Execute returns `{ action: 'add_to_cart', product: { id, name, price }, quantity, message: 'Added to cart!' }` (simulated).
