    3. Create `src/components/motion/ScrollReveal.tsx`:
       - "use client" directive
       - Migrate from `src/components/common/ScrollReveal.tsx` in Vite codebase
       - Change import from `framer-motion` to `motion/react`
       - Props: children, delay (default 0), direction ('up' | 'left' | 'right' | 'none'), className

       - Use `useReducedMotion()` from motion/react to skip animations when user prefers reduced motion

       - viewport={{ once: true, margin: '-50px' }} to trigger slightly before element enters view

    4. Create `src/components/motion/AnimatePresenceWrapper.tsx`:
       - "use client" directive
       - Re-export `AnimatePresence` from `motion/react`
       - Props: children, mode ('wait' | 'sync' | 'popLayout', default 'wait')
       - Thin wrapper so Server Components can use exit animations

    5. Create `src/hooks/useReducedMotion.ts`:
       - "use client" directive
       - Re-export `useReducedMotion` from `motion/react` for convenience
       - This allows other client components to check reduced motion preference
